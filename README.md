# Implementasi-Algoritma-Genetika-Genetic-Algorithm-untuk-Pencarian-Kata

# Fungsi program
Program ini merupakan implementasi Algoritma Genetika (Genetic Algorithm) untuk melakukan pencarian kata pada kamus Bahasa Selayar. Program bekerja dengan menghasilkan populasi awal berupa kromosom acak, kemudian melakukan proses seleksi, crossover, dan mutasi secara berulang hingga memperoleh kata yang memiliki tingkat kemiripan (fitness) tertinggi terhadap kata yang dicari.

# Cara menjalakan
python3 kamus_selayar.py

# Fitur - fitur
1. Tampilkan Kamus
2. Cari Kata
3. Jalankan Algoritma Genetika
4. Tampilkan Populasi
5. Hasil Fitness
6. Seleksi Roulette
7. Cross Over
8. Mutasi
9. Generasi Baru
10. Keluar
